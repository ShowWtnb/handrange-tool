# Poker Tools

## Overview
このツールは[ヨコサワポーカーチャンネル](https://www.youtube.com/@yokosawa_poker)の解説動画をもとに動画の内容を解釈し、個人的な利用を目的に作成したものです

THIS TOOL IS ONLY BASED ON THE CONTENT OF THE [ヨコサワポーカーチャンネル](https://www.youtube.com/@yokosawa_poker)'s VIDEO.
